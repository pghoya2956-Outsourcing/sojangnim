# 제품 페이지 디자인 시안 모음

공구/공업 제품을 위한 7가지 웹 페이지 디자인 시안입니다. 각 시안은 다른 컨셉과 사용자 경험을 제공합니다.

## 시안 목록

### 01. Minimalist Industrial (미니멀 산업용)
**파일:** `01-minimalist-industrial.html`

**컨셉:**
- 여백과 타이포그래피를 강조한 미니멀 디자인
- 흰색 배경에 제품이 돋보이는 깔끔한 레이아웃
- 불필요한 요소를 최소화하여 제품에 집중

**특징:**
- 심플한 색상 팔레트 (흰색, 검은색, 회색)
- 큰 제품 이미지 영역 (400px 높이)
- 여백을 활용한 공간감
- 레터스페이싱과 대문자로 전문적인 느낌

**적합한 사용처:**
- 고급 산업용 공구 브랜드
- 미니멀리즘을 추구하는 브랜드
- 제품 자체에 집중하고 싶을 때

---

### 02. Compact Dense (컴팩트 덴스)
**파일:** `02-compact-dense.html`

**컨셉:**
- 한 화면에 많은 정보를 효율적으로 표시
- 실용적이고 기능 중심의 디자인
- 오렌지 포인트 컬러로 시선 유도

**특징:**
- 작은 카드 크기로 한 화면에 더 많은 제품 표시
- 제품 코드, 스펙, 재고 상태를 한눈에 확인
- 다크 헤더에 오렌지 액센트 (FF8800)
- 컴팩트한 레이아웃 (200px 이미지 높이)

**적합한 사용처:**
- 제품 종류가 많은 도매 업체
- 빠른 탐색과 비교가 필요한 B2B 쇼핑몰
- 카탈로그 형식의 제품 검색

---

### 03. Blueprint Technical (블루프린트 테크니컬)
**파일:** `03-blueprint-technical.html`

**컨셉:**
- 공학 도면(blueprint)에서 영감을 받은 디자인
- 기술적이고 전문적인 분위기
- 파란색 계열의 독특한 색상 팔레트

**특징:**
- 다크 블루 배경 (#0a2540)에 시안 텍스트
- 그리드 패턴 오버레이로 도면 느낌
- 스펙을 테이블 형식으로 명확하게 표시
- 네온 효과와 그림자로 미래지향적 느낌
- Courier 폰트로 기술 문서 스타일

**적합한 사용처:**
- 엔지니어링 전문 업체
- 기술 중심 제품 브랜드
- 차별화된 브랜드 아이덴티티를 원할 때
- 밤에 보기 편한 다크 테마 선호 시

---

### 04. Modern Box (모던 박스)
**파일:** `04-modern-box.html`

**컨셉:**
- 현대적이고 세련된 박스 레이아웃
- 제품 상세 정보를 구조화된 형태로 제공
- 사용자 친화적인 인터페이스

**특징:**
- 라운드 코너와 그림자로 입체감
- 회색 박스로 스펙을 구분하여 가독성 향상
- 카테고리 라벨과 구조화된 정보 표시
- 장바구니 버튼으로 실제 쇼핑몰 느낌
- 호버 시 부드러운 애니메이션

**적합한 사용처:**
- 현대적인 이커머스 사이트
- B2C 타겟의 공구 쇼핑몰
- 사용자 경험을 중시하는 플랫폼
- 실제 구매까지 연결되는 온라인 스토어

---

### 05. Catalog List (카탈로그 리스트)
**파일:** `05-catalog-list.html`

**컨셉:**
- 전통적인 카탈로그를 웹으로 구현
- 가로형 리스트 뷰로 상세 정보 표시
- 제품 비교와 상세 확인에 최적화

**특징:**
- 제품을 리스트 형태로 나열 (카드가 아닌 행)
- 왼쪽에 큰 제품 이미지 (200x200px)
- 오른쪽에 상세 설명, 스펙, 가격, 주문 버튼 배치
- 배지(신제품, 베스트, 프리미엄)로 제품 강조
- 카테고리별 제품 수 표시

**적합한 사용처:**
- 전문 B2B 카탈로그 사이트
- 상세한 제품 비교가 필요한 경우
- 인쇄 카탈로그와 유사한 경험 제공
- 대량 구매를 위한 견적 사이트

---

### 06. Table View (테이블 뷰)
**파일:** `06-table-view.html`

**컨셉:**
- 데이터베이스 테이블 형식의 효율적인 정보 표시
- 재고 관리 시스템처럼 정보 중심
- 빠른 스캔과 비교에 최적화

**특징:**
- HTML 테이블 기반 레이아웃
- 제품명, 사양, 가격, 재고, 액션이 컬럼으로 구분
- 재고 상태를 배지로 시각화
- 호버 시 행 강조
- 그리드/테이블 뷰 토글 버튼

**적합한 사용처:**
- ERP 시스템 연동 사이트
- 재고 관리가 중요한 B2B 플랫폼
- 대량 주문 처리 시스템
- 관리자 또는 딜러용 인터페이스

---

### 07. Professional Clean (프로페셔널 클린)
**파일:** `07-professional-clean.html`

**컨셉:**
- 깔끔하고 전문적인 기업용 디자인
- 신뢰감을 주는 정돈된 레이아웃
- 고급스러운 느낌의 심플한 디자인

**특징:**
- 흰색 배경에 검은색 텍스트로 가독성 극대화
- 카테고리, 제품명, 설명, 스펙이 명확히 구분
- 밑줄 링크로 전통적이고 신뢰감 있는 UI
- 여백과 타이포그래피로 품격 표현
- Sticky 헤더로 네비게이션 접근성

**적합한 사용처:**
- 대기업 또는 글로벌 브랜드
- 전문 솔루션 제공 업체
- 신뢰와 품질을 강조하고 싶을 때
- 보수적인 고객층 타겟

## 공통 특징

모든 시안은 다음을 포함합니다:

- **상단 네비게이션**: 로고, 메뉴, 로그인 버튼
- **좌측 사이드바**: 카테고리 필터 메뉴
- **제품 그리드/리스트**: 제품 카드 또는 행
- **제품 정보**: 이름, 설명, 가격, 스펙
- **반응형 고려**: 그리드 레이아웃으로 화면 크기 적응

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션
- **순수 CSS**: 외부 라이브러리 없이 구현

## 커스터마이징 포인트

각 시안에서 쉽게 변경 가능한 요소:

- **색상**: CSS 변수로 브랜드 컬러 적용

- **간격**: padding, margin, gap 조정
- **이미지 크기**: height 속성 수정
