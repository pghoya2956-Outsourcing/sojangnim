# 소장님 제품 카탈로그 웹사이트

## 이 웹사이트가 필요한 이유

지금까지 고객에게 제품을 소개하고 견적서를 전달하려면:
1. 제품 카탈로그 PDF나 사진을 일일이 전송하거나
2. 엑셀로 견적서를 직접 작성해서 보내거나
3. 전화로 가격을 안내하는 번거로운 과정이 필요했습니다.

**이 웹사이트는 이 모든 과정을 자동화합니다.**

---

## 핵심 기능

### 1. 온라인 제품 카탈로그
- **모든 제품 정보를 한곳에**: 제품명, 가격, 상세 사양, 이미지
- **카테고리별 분류**: 전동 공구, 수공구, 측정기 등 체계적으로 정리
- **배지 표시**: 신제품, 베스트, 프리미엄, 할인 제품을 한눈에 확인

### 2. 견적서 자동 생성
- **고객이 직접 선택**: 원하는 제품을 장바구니에 담기
- **수량 조절 가능**: 각 제품의 수량을 자유롭게 변경
- **즉시 출력**: 버튼 클릭 한 번으로 견적서 PDF 생성
- **총액 자동 계산**: 수량 × 가격 자동 계산, 전체 합계 표시

### 3. 24시간 접근 가능
- **언제든지 확인**: 고객이 원하는 시간에 제품 정보 확인
- **회원가입 불필요**: 별도 로그인 없이 바로 이용 가능
- **모바일 지원**: 스마트폰, 태블릿, PC 모두 사용 가능

---

## 사용 방법 (고객 입장)

### Step 1: 웹사이트 접속
```
웹사이트 주소를 입력하면 바로 제품 목록을 확인할 수 있습니다.
(별도의 회원가입이나 로그인이 필요 없습니다)
```

### Step 2: 제품 둘러보기
- **홈페이지**: 추천 제품과 최신 제품을 먼저 확인
- **제품 목록**: 왼쪽 메뉴에서 카테고리 선택
- **제품 상세**: 제품을 클릭하면 자세한 사양과 이미지 확인

### Step 3: 장바구니에 담기
- 원하는 제품의 **"장바구니 담기"** 버튼 클릭
- 상단 우측의 장바구니 아이콘에 숫자 표시 (담긴 제품 개수)

### Step 4: 견적서 받기
- 우측 상단 **장바구니 아이콘** 클릭
- 수량 조절 (+ / - 버튼 또는 직접 입력)
- 필요 없는 제품은 삭제 가능
- **"견적서 출력"** 버튼 클릭 → PDF로 저장 또는 인쇄

## 이렇게 활용하세요

### 활용 사례 1: 고객 상담 시
```
전화 통화 중 → "홈페이지에서 XX 제품 보시면 됩니다"
→ 고객이 직접 확인 후 장바구니에 담기
→ 견적서 즉시 출력
```

### 활용 사례 2: 명함이나 카톡에 주소 공유
```
"저희 제품은 여기서 확인하세요: [웹사이트 주소]"
→ 고객이 시간 날 때 편하게 둘러보기
→ 관심 제품을 장바구니에 담아두고 나중에 연락
```

### 활용 사례 3: 단골 고객 재주문
```
이전에 주문했던 제품을 다시 찾아서
→ 장바구니에 담고 견적서 출력
→ 견적서 캡처해서 카톡으로 전송
```

> 실수로 데이터가 삭제되지 않도록 안전하게 설계되었습니다.

- ❌ 전: 엑셀로 견적서 작성 (10분)
- ✅ 후: 고객이 직접 견적서 생성 (1분)

- 24시간 언제든지 제품 확인 가능
- 직접 원하는 제품을 선택하는 편리함
- 깔끔하고 전문적인 인상

### 🔄 유지보수 용이
- 제품 정보 실시간 업데이트
- 가격 변경 즉시 반영
- 재고 없는 제품 숨김 처리 가능

## 자주 묻는 질문 (FAQ)

### Q1. 고객이 직접 주문할 수 있나요?
**A.** 아니요. 이 웹사이트는 **견적서 생성 전용**입니다.
- 고객이 장바구니에 제품을 담고 견적서를 출력
- 출력된 견적서를 소장님께 전달 (카톡, 이메일 등)
- 소장님이 확인 후 정식 주문 진행

### Q2. 제품 정보는 어떻게 변경하나요?
**A.** 관리자 페이지에서 간단하게 수정 가능합니다.
- 제품 추가/삭제/수정
- 가격 변경
- 이미지 교체
- 상세 사양 업데이트

### Q3. 고객 정보를 수집하나요?
**A.** 아니요. 회원가입이 없으므로 **고객 정보를 전혀 수집하지 않습니다**.
- 장바구니 정보는 고객의 브라우저에만 저장
- 개인정보 보호 걱정 없음

### Q4. 모바일에서도 잘 보이나요?
**A.** 네. 스마트폰, 태블릿, PC 모두 최적화되어 있습니다.
- 반응형 디자인 적용
- 어떤 화면 크기에서도 깔끔하게 표시

### Q5. 비용은 얼마나 드나요?
**A.** 웹사이트 호스팅을 위한 **월 서버 비용**이 발생합니다.

**비용 구성:**
- **Vercel (웹사이트 호스팅)**: 월 $20 (약 27,000원)
- **Supabase (데이터베이스)**: 월 $25 (약 34,000원)
- **합계**: 월 $45 (약 **60,000원**)

**비용 설명:**
- 이 비용은 웹사이트를 24시간 안정적으로 운영하기 위한 서버 임대료입니다
- 방문자 수가 적어도 동일한 비용이 발생합니다 (정액제)
- 별도의 개발 비용이나 유지보수 비용은 없습니다
- 제품 정보 수정이나 기능 개선은 추가 비용 없이 가능합니다

> **참고**: 개발자(저)는 수익을 목적으로 하지 않으며,
> 실제 서버 운영에 필요한 비용만 부담해주시면 됩니다.
