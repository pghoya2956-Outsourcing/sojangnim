## 제1조 (목적)

본 계약은 "갑"이 "을"에게 제공하는 웹사이트 구축 및 호스팅 서비스의 이용 조건과 절차, 권리와 의무를 규정함을 목적으로 한다.

---

## 제2조 (서비스 내용)

### 1. 웹사이트 구축 서비스

"갑"은 "을"에게 다음의 웹사이트 구축 서비스를 제공한다.

- [ ] 제품 카탈로그 웹사이트 구축
- [ ] 장바구니 및 견적서 출력 기능
- [ ] 반응형 디자인 (PC/모바일)
- [ ] 제품 등록 ____ 개 대행
- [ ] 커스텀 도메인 연결
- [ ] 디자인 커스터마이징 (색상/로고)
- [ ] 사용법 교육 (____분)
- [ ] 기타: ____________________

### 2. 호스팅 서비스 (선택)

- [ ] 월정액 호스팅 서비스 이용
- [ ] 셀프 호스팅 (구축 후 "을"이 직접 관리)

호스팅 서비스 이용 시 "갑"은 다음을 제공한다:
- 웹사이트 서버 호스팅
- 주간 자동 백업
- 월 1시간 이내 소소한 수정 지원
- 장애 발생 시 24시간 내 대응

## 제3조 (서비스 비용)

### 1. 구축 비용

| 항목 | 금액 |
|------|------|
| 기본 구축비 | ____________ 원 |
| 추가 옵션 | ____________ 원 |
| **합계** | **____________ 원** |

### 2. 호스팅 비용 (해당 시)

- 월 호스팅 비용: ____________ 원
- 결제 주기: 월납 / 연납
- 연납 시 할인: ____________ 원/월

### 3. 결제 조건

**구축비**
- 계약금: 구축비의 50% (계약 시)
- 잔금: 구축비의 50% (구축 완료 후)

**호스팅비**
- 결제일: 매월 ____ 일
- 결제 방법: 계좌이체 / 카드 자동결제
- 입금 계좌: ____________ 은행 ____________

## 제4조 (작업 기간)

- 작업 시작일: 계약금 입금 확인 후
- 예상 완료일: 착수일로부터 ____ 영업일
- 최종 완료: "을"의 검수 승인 후

---

## 제5조 (권리와 의무)

### 1. "갑"의 의무

1. 합의된 서비스를 성실히 제공한다.
2. 호스팅 서비스의 경우 서버 가용성 99% 이상을 유지한다.
3. "을"의 데이터를 안전하게 관리하고 제3자에게 제공하지 않는다.
4. 장애 발생 시 24시간 내 대응한다.

### 2. "을"의 의무

1. 서비스 비용을 기한 내 납부한다.
2. 구축에 필요한 자료(제품 정보, 이미지 등)를 적시에 제공한다.
3. 서비스를 불법적 목적으로 사용하지 않는다.
4. 관리자 계정 정보를 안전하게 관리한다.

---

## 제6조 (수정 및 추가 개발)

### 1. 무상 수정 범위

- 구축 완료 후 ____ 개월 이내
- 버그 수정
- 텍스트 오타 수정
- 호스팅 고객: 월 1시간 이내 소소한 수정

### 2. 유상 수정 범위

다음 작업은 별도 비용이 발생한다:
- 신규 기능 개발
- 대규모 디자인 변경
- 제품 등록 대행
- 외부 서비스 연동

---

## 제7조 (호스팅 서비스 해지)

### 1. "을"의 해지

- 1개월 전 서면(이메일 포함) 통보
- 기납부 호스팅비는 환불하지 않음
- 해지 시 데이터 백업본 제공

### 2. "갑"의 해지

- 서비스 종료 1개월 전 통보
- 데이터 백업본 및 소스코드 제공
- 다른 호스팅으로 이전 지원

### 3. 미납에 의한 해지

- 결제일로부터 7일 경과: 1차 안내
- 결제일로부터 14일 경과: 서비스 중단 경고
- 결제일로부터 30일 경과: 서비스 중단
- 결제일로부터 60일 경과: 계약 자동 해지, 데이터 삭제

---

## 제8조 (지적 재산권)

1. 구축 완료된 웹사이트의 소유권은 "을"에게 귀속된다.
2. 단, "갑"이 개발한 공통 컴포넌트 및 라이브러리의 저작권은 "갑"에게 귀속된다.
3. "갑"은 구축 사례를 포트폴리오로 활용할 수 있다. (거부 시 별도 협의)

---

## 제9조 (비밀 유지)

양 당사자는 본 계약 이행 과정에서 알게 된 상대방의 비밀 정보를 계약 기간 및 종료 후 2년간 제3자에게 공개하지 않는다.

---

## 제10조 (면책)

"갑"은 다음의 경우 책임을 지지 않는다:

1. 천재지변, 전쟁 등 불가항력에 의한 서비스 중단
2. "을"의 귀책사유로 인한 손해
3. "을"이 제공한 콘텐츠의 저작권 문제
4. 호스팅 업체(Vercel, Supabase 등)의 장애

---

## 제11조 (분쟁 해결)

본 계약과 관련한 분쟁은 "갑"의 소재지를 관할하는 법원을 합의 관할로 한다.

## 별첨: 서비스 상세 사양

### 기술 스택

- 프론트엔드: Next.js, React, TypeScript
- 스타일링: Tailwind CSS
- 데이터베이스: Supabase (PostgreSQL)
- 호스팅: Vercel

### 제공 기능

1. 제품 카탈로그
   - 카테고리별 제품 조회
   - 제품 상세 정보 표시
   - 제품 사양(스펙) 표시

2. 장바구니
   - 제품 담기/삭제
   - 수량 조절
   - 로컬 저장 (브라우저)

3. 견적서
   - 장바구니 기반 견적서 생성
   - 인쇄용 레이아웃
   - PDF 저장 가능

4. 관리자 기능
   - Supabase Dashboard를 통한 제품 관리
   - 카테고리 관리

### 미포함 기능

- 결제/주문 시스템
- 회원가입/로그인
- 재고 관리
- 배송 추적
